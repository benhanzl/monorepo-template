# monorepo-template

A minimal pnpm monorepo template.

## Requirements

- Node.js 24.12.0
- pnpm 10.27.0

## Structure

```
monorepo-template/
├── apps/          # Application packages
├── packages/      # Shared library packages
├── package.json   # Root workspace configuration
└── pnpm-workspace.yaml
```

## Getting Started

```bash
# Install dependencies
pnpm install
```
